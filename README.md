# TodoApp

A simple task management app built with **Vite**, **React**, and **JavaScript**. Manages tasks with statuses like Pending, In Progress, and Done.

## Features

- Responsive design for all devices.
- Reusable React components.
- Dynamic styles for task titles and statuses.

## Components

- **Header**: Displays "TodoApp".
- **Subheader**: Displays "Todo List Manager".
- **ListHeader**: Displays "Todo List".
- **ItemCard**: Displays task details with conditional styling.
- **ItemsList**: Renders a list of tasks.
- **ListContainer**: Wraps the list and its header.

## Deployment

The app is deployed and accessible via a preview URL.
